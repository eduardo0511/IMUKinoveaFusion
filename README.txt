# Repositorio de Investigación

## Descripción
Este repositorio contiene el código Arduino utilizado para la recopilación de datos, 
los datos recolectados por el sensor IMU de varios sujetos y los videos capturados para su análisis en Kinovea. 
La investigación se enfoca en un analisis cinematico del gesto tecnico del pase en futbolistas universitarios.

En este repositorio, encontrarás los siguientes elementos:

### 1. Código Arduino
- El código fuente de Arduino utilizado para la recopilación de datos con el sensor IMU. 
Puede encontrarse en la carpeta Cod_final.

### 2. Datos IMU
- Los datos brutos recolectados por el sensor IMU de cada sujeto en formato CSV. 
Los datos están organizados en carpetas por sujeto.

### 3. Videos de Análisis en Kinovea
- Los videos capturados para el análisis en el software Kinovea. 
Los videos están organizados en carpetas por sujeto.

### 4. Consentimiento Informado
Dentro de las carpetas correspondientes, encontrarás el formulario de consentimiento informado,
el cual fue proporcionado y firmado por cada sujeto que participó en el estudio.
Este documento es esencial para garantizar la ética y la integridad de la investigación.
